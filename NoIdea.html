Endless Runner

This endless runner is a small game that I made for my portfolio.

What kind of game is it.

This game is an endless runner, like crossy road where you need to jump on logs and avoid cars to get the highest score possible. As a bonus you can also collect coins.

Why did I make this?

I made this so I can learn how to make basic games and write basic scripts on my own.
