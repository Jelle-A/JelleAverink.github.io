Dungeon Crawler

This Dungeon Crawler is a small game that I made for my portfolio.

What kind of game is it.

This game is like and Dungeon Crawler where need to escape by going up, you can view this as an simplified jump knight. In this small game you can slao collect coins and other goodies. You also can die by the enemies so watch out!

Why did I make this?

I made this so I can learn how to use my own assets in small Unity games and make these assets interact with eachother. I also made it so I can improve on my scripting skills.
